ТЗ по коду
1. Не забываем за структуру html5. Подумайте над тегами, которые вы юзаете в той или иной ситуации
2. Макет нужно сверстать по максимум используя flexbox ( там где он нужен )
3. Использовать в работе один из препроцессоров на выбор ( stylus, sass )
3.1 Вложенность sass/styl кода не должна быть более 3-х
3.2 Ваш sass/styl код разбить на структуру ( не писать все в 1 файле ). 
https://www.dropbox.com/s/ggh1hacabqy5n8y/styles.zip?dl=0 - пример папки styles с минимальной структурой, которую определили на уроке
3.3 Для глобальных значений ( цвета, фонт сайзы и тд ) юзайте переменные в препроцессоре
